# Sushi King Homepage

A pixel-perfect responsive homepage for Sushi King built with Next.js, React, and Tailwind CSS.

## Features

- **Responsive Design**: Mobile-first approach with hamburger menu for small screens
- **Modern UI**: Clean, minimalist design with custom wave background
- **Interactive Elements**: Hover effects and smooth transitions
- **Optimized Images**: Next.js Image component for performance
- **TypeScript**: Full type safety throughout the application

## Design Elements

- **Navigation Bar**: Logo with fish symbol, responsive menu
- **Hero Section**: Wavy background pattern, centered content, circular food images
- **Address Section**: Two-column layout with location pins
- **Footer**: Navy background with location information

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Tech Stack

- **Next.js 14**: React framework
- **TypeScript**: Type safety
- **Tailwind CSS**: Utility-first CSS framework
- **Inter Font**: Modern sans-serif typography

## Customization

- Colors are defined in `tailwind.config.js`
- Wave background pattern is created with CSS and SVG
- Images are placeholder URLs that can be replaced with actual assets
